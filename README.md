### Features

- User Authentication: Simple login system for admin/warden access.
- Room Management: Add, update, delete, search, and view all rooms.
- Student Management: Add, update, delete, search, and view all students.
- Bed Allocation: Allocate and vacate beds automatically, with occupancy tracking.
- Transfers: Move students between rooms with automatic updates of available beds.
- Reports: Occupancy grid, Vacant beds by floor, Students per room, Overdue dues and fines, Daily revenue projections
- Validations: Ensures uniqueness, correct input parsing, and occupancy rules.
